## 🚀 触发 GitHub Actions 打包

### 步骤 1：打开 GitHub 仓库

访问：https://github.com/msyzdj-web/property-manager-win7

### 步骤 2：进入 Actions 页面

- 点击顶部的 **"Actions"** 标签页
- 如果提示需要启用 Actions，请点击启用

### 步骤 3：选择工作流

在工作流列表中找到：
**"Build Windows 7 Compatible EXE"**

### 步骤 4：运行工作流

1. 点击 **"Run workflow"** 按钮
2. 选择分支（通常是 `main`）
3. 点击绿色的 **"Run workflow"** 按钮

### 步骤 5：等待打包

- 工作流状态会显示 "in progress"
- 打包过程需要 **5-10 分钟**
- 所有步骤完成后会显示绿色 ✓

### 步骤 6：下载结果

1. 点击进入完成的工作流运行记录
2. 滚动到页面底部
3. 找到 **"Artifacts"** 部分
4. 点击 **"PropertyManager-Win7-EXE"** 下载
5. 解压下载的文件
6. 获得 `物业收费管理系统_Win7.exe`

# 文件保存位置说明

## 导出文件保存位置

### 默认保存位置
所有导出的Excel文件（欠费清单、缴费记录、统计报表）默认保存在：

```
程序目录/exports/
```

例如：
- 如果程序在 `E:\桌面\wykf\`，则导出文件保存在 `E:\桌面\wykf\exports\`
- 如果程序在 `C:\Program Files\物业管理系统\`，则导出文件保存在 `C:\Program Files\物业管理系统\exports\`

### 文件命名规则

1. **欠费清单**
   - 文件名：`欠费清单_YYYY-MM.xlsx`
   - 示例：`欠费清单_2024-01.xlsx`

2. **缴费记录**
   - 文件名：`缴费记录_YYYYMMDD.xlsx`
   - 示例：`缴费记录_20240115.xlsx`

3. **统计报表**
   - 文件名：`月度统计报表_YYYY-MM.xlsx`
   - 示例：`月度统计报表_2024-01.xlsx`

### 自定义保存位置

导出时，系统会：
1. 默认打开 `exports` 文件夹
2. 显示默认文件名
3. 您可以：
   - 直接保存到默认位置（点击保存）
   - 选择其他文件夹保存（浏览到其他位置）
   - 修改文件名

### 备份文件保存位置

数据备份文件保存在：

```
程序目录/backup/
```

备份文件名格式：`property_backup_YYYYMMDD_HHMMSS.db`

---

## 目录结构

```
程序目录/
├── main.py
├── property.db          # 数据库文件
├── exports/            # 导出文件目录（自动创建）
│   ├── 欠费清单_2024-01.xlsx
│   ├── 缴费记录_20240115.xlsx
│   └── 月度统计报表_2024-01.xlsx
└── backup/             # 备份文件目录（自动创建）
    ├── property_backup_20240115_120000.db
    └── property_backup_20240115_180000.db
```

---

## 注意事项

1. **自动创建目录**：`exports` 和 `backup` 文件夹会在首次使用时自动创建
2. **文件覆盖**：如果文件名已存在，系统会提示是否覆盖
3. **文件访问**：导出的Excel文件可以直接用Excel、WPS等软件打开
4. **文件管理**：建议定期整理导出文件，避免文件过多

---

## 快速访问

导出完成后，系统会显示文件保存路径，您可以：
1. 点击消息框中的路径（如果支持）
2. 或手动打开 `exports` 文件夹查看文件

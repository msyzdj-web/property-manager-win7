# 物业收费管理软件项目开发完整计划书

## 一、项目概述

本项目旨在开发一套本地运行的物业收费管理软件，用于小型物业场景，实现住户管理、收费、欠费查询与收据打印。

## 二、项目目标

-   实现标准化物业收费流程
-   减少人工记账错误
-   支持本地长期稳定运行
-   操作简单，易于上手

## 三、功能范围

### 3.1 核心功能

-   住户管理
-   收费项目管理
-   收费与缴费记录
-   欠费查询
-   收据打印
-   月度统计

### 3.2 非功能需求

-   本地 SQLite 数据存储
-   数据可备份
-   稳定、易维护

## 四、技术方案

-   Python 3.10+
-   PySide6
-   SQLite + SQLAlchemy
-   PyInstaller 打包

## 五、系统架构

UI 层 → Service 层 → Model 层 → SQLite 数据库

# 快速启动指南

## 一、环境准备

1. **安装Python**
   - 确保已安装Python 3.10或更高版本
   - 检查命令：`python --version`

2. **安装依赖包**
   ```bash
   pip install -r requirements.txt
   ```
   或手动安装：
   ```bash
   pip install PyQt5==5.15.10
   pip install SQLAlchemy==2.0.23
   pip install pyinstaller==6.2.0
   ```

## 二、运行程序

### 方式一：直接运行Python脚本
```bash
python main.py
```

### 方式二：打包为exe后运行
```bash
# Windows系统
build.bat

# 或手动打包
pyinstaller -F -w main.py
```

打包完成后，exe文件位于 `dist` 目录下。

## 三、首次使用

1. **启动程序**
   - 首次运行会自动创建数据库文件 `property.db`

2. **添加住户**
   - 点击"住户管理"标签页
   - 点击"新增住户"按钮
   - 填写房号（必填，唯一）、姓名（必填）、电话、面积等信息
   - 点击"保存"

3. **添加收费项目**
   - 点击"收费项目管理"标签页
   - 点击"新增收费项目"按钮
   - 填写项目名称、选择收费类型（固定/按面积/手动）、设置单价/金额
   - 点击"保存"

4. **生成账单**
   - 点击"收费管理"标签页
   - 点击"生成账单"按钮
   - 选择住户、收费项目、缴费周期
   - 系统自动计算金额（手动类型需手动输入）
   - 点击"生成账单"

5. **标记缴费**
   - 在"收费管理"标签页选择未缴费的记录
   - 点击"标记已缴费"按钮

6. **打印收据**
   - 选择已缴费的记录
   - 点击"打印收据"按钮
   - 预览收据内容
   - 点击"打印"按钮进行打印

7. **查询欠费**
   - 点击"欠费查询"标签页
   - 选择查询周期
   - 查看未缴费清单和统计信息

## 四、数据备份

- 复制 `property.db` 文件即可完成数据备份
- 建议定期备份数据库文件

## 五、常见问题

### 1. 程序无法启动
- 检查Python版本是否符合要求
- 检查是否已安装所有依赖包
- 查看错误提示信息

### 2. 房号重复
- 房号必须唯一，如果提示重复，请检查是否已存在该房号

### 3. 无法删除住户/收费项目
- 如果该住户/收费项目有关联的缴费记录，需要先删除相关记录才能删除

### 4. 打印功能无法使用
- 确保系统已安装打印机驱动
- 检查打印机是否正常工作

## 六、技术支持
